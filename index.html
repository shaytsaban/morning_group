מבוא למערכות מידע וניהול נתונים

קורס מקיף המחבר בין עקרונות ניהול מידע לבין טכנולוגיות המידע העדכניות ביותר. מתאים لكل מי שרוצה להבין איך עובדים מערכות מידע, כיצד מאוחסנים נתונים, ואיך ניתן לנתח אותם.

על הקורס

הקורס "מבוא למערכות מידע וניהול נתונים" נועד לאפשר לסטודנטים להבין את היסודות של מערכות מידע, ניהול נתונים, אבטחת מידע, יישומי BI (Business Intelligence),大数据, Cloud Computing, Data Warehousing ועוד.

במהלך הקורס תפגשו כלים מתקדמים כמו SQL, Excel, Power BI, Tableau, MongoDB ועוד.

תוכנית הלימודים

  • מבוא למערכות מידע
  • בסיסי נתונים ומודלים נתונים
  • שפת SQL וניהול נתונים
  • אחסון מידע בענן
  • אבטחת מידע ופרטיות נתונים
  • ניתוח נתונים בעזרת BI Tools
  • 大数据 ו-AI בסביבות מידע

מגמות מודרניות

ניתוח海量 נתונים בעזרת כלים מתקדמים.

Data Science & AI
